## Contributing Figures

**Guidelines:**
- Use SVG format (vector graphics, scalable)
- High contrast for readability
- Include source files (if created in Figma, draw.io, etc.)
- Add description in this README

**Tools we recommend:**
- draw.io (free, web-based)
- Figma (free tier available)
- Inkscape (free, open source)
- Python matplotlib (for data figures)
